- フォルダの構造
    - input
        - 各モデルの特徴量など、学習に必要なインプットを入れています。kaggle datasetで共有しています。
    - output
        - 各モデルの予測値と、ensembleした最後の結果です。kaggle datasetで共有しています。
    - model
        - 学習済みモデルです。kaggle datasetで共有しています。
    - code
        - 各featureを作成するcodeとmodelを再現するcodeです。
          kaggle notebookで動くようにしています。
